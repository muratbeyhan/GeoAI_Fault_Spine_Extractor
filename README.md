# GeoAI: Fault Spine Extractor
This QGIS plugin extracts structural fault spines from earthquake catalogs using PCA and DBSCAN clustering.

### Key Features:
- **Seismic Analytics:** Automatic calculation of **Strike** and **Segment Length**.
- **Adaptive Smoothing:** Precise control over fault geometry.
- **Field Mapping:** Support for various catalog formats (AFAD, JMA, USGS).
